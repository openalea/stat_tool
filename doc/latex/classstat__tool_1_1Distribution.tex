\doxysubsection{Detailed Description}
Discrete distribution. 

Probabilities are represented as an array \texorpdfstring{$\ast$}{*}mass, with size nb\+\_\+allocated\+\_\+value mass\mbox{[}i\mbox{]} is only meaningful for offset \texorpdfstring{$<$}{<}= i \texorpdfstring{$<$}{<} nb\+\_\+value. For i \texorpdfstring{$<$}{<} offset or i \texorpdfstring{$>$}{>}= nb\+\_\+value, mass\mbox{[}i\mbox{]} may be either 0 or unspecified. nb\+\_\+allocated\+\_\+value has to be greater or equal to nb\+\_\+value. It may happen that nb\+\_\+allocated\+\_\+value \texorpdfstring{$>$}{>} nb\+\_\+value, particularly when mass\mbox{[}i\mbox{]} is computed for every i \texorpdfstring{$<$}{<}= quantile(0.\+9999) = nb\+\_\+value-\/1, which we do not know in advance, but we allocated nb\+\_\+allocated\+\_\+value \texorpdfstring{$>$}{>} nb\+\_\+value for the sake of safety.
